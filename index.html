HTML

HTML é um acrônimo para HiperText Markup Language, que traduzido para o português significa Linguagem de Marcação de Hipertexto. Foi criado pelo britânico Tim Berners-Lee e é o componente básico da web, permitindo inserir o conteúdo e estabelecer a estrutura básica de um website.

CSS

O CSS (Cascading Style Sheets) é uma linguagem de estilo que define a aparência de páginas web, como cores, fontes e layout. Ele separa o conteúdo da estrutura visual, facilitando a criação de sites elegantes e consistentes.

JavaScript

JavaScript é uma linguagem de programação de alto nível, interpretada e orientada a objetos, amplamente utilizada para desenvolver aplicações web. Ela permite a criação de interatividade e dinamismo em sites, tornando a experiência do usuário mais rica e envolvente. JavaScript é uma das principais tecnologias da web, junto com HTML e CSS, e é utilizada na maioria dos sites para implementar funcionalidades complexas, como atualizações de conteúdo em tempo real e gráficos animados.
